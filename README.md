# XGecu_Software
This is a mirror of current and archival versions of software for XGecu's (formerly Autoelectric's) chip programmers, both to make older versions available if needed, as well as to provide a source which is somewhat faster than the official site.  The set of programmers currently supported by the software here includes the TL886A, the TL866CS, the TL866II Plus, the T48, the T56, and the T76.

Note that this archive is missing many versions of the TL866A/TL866CS software (minipro_setupXXX.rar).  If anybody has copies of missing versions, please submit them.

As of versions 13.03 and 13.04, Xgecu seems to be putting the model numbers in the filename which the software applies to.  Currently, that split seems to be between the TL866II+/T48/T56 (included in one installer) and the T76 (included in a different installer).

Also, note that the TL866II+ and T56 are listed with the word "STOP" by them on [the official download site](http://www.xgecu.com/EN/download.html), just like the original TL866A/CS products have.  The TL866II+ and T56 do still seem to be receiving updates, but it's unclear how long that will continue.
